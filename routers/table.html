טבלאות API

טבלת API של משתמשים
CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET Api/user - - מחזיר את כל המשתמשים 400: שגיאה כללית בשרת
GET Api/user/id id - מחזיר משתמש לפי מזהה 404: משתמש לא נמצא, 400: מזהה לא חוקי
PUT Api/user/id שדות לעדכון - מעדכן משתמש לפי מזהה 404: משתמש לא נמצא, נתונים לא חוקיים, 400: בעיה בשרת
PUT Api/user/Update_password email, newPassword - עדכון סיסמא 404: אימייל לא נמצא, 400: בעיה בשרת
POST Api/user phone, email, username, password - הוספת משתמש 404: נתונים חסרים, 400: בעיה בשרת
POST Api/user/login username, password - שליפת משתמש לפי שם וסיסמא 404: שם משתמש או סיסמא לא נכונים, 400: בעיה בשרת
טבלת API של חדרים
CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET Api/room/ - - מחזירה את כל החדרים 400: שגיאה כללית בשרת
GET Api/room/id id - מחזירה חדר לפי מספר מזהה 404: חדר לא נמצא, 400: מזהה לא חוקי
DELETE Api/room/id id - מחיקת חדר לפי מספר מזהה 404: חדר לא נמצא, 400: מזהה לא חוקי
PUT Api/room/id num, numBads - עדכון חדר לפי מזהה 404: חדר לא נמצא, נתונים לא חוקיים, 400: בעיה בשרת
POST Api/room num, numBads - הוספת חדר 404: נתונים חסרים, ערכים לא חוקיים, 400: בעיה בשרת
טבלת API של הזמנות
CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET Api/invitations - - מחזיר את כל ההזמנות 400: שגיאה כללית בעת שליפת ההזמנות
GET Api/invitations/userid userid - מחזיר את כל ההזמנות לפי מזהה של משתמש 400: שגיאה כללית בעת שליפת ההזמנות לפי מזהה משתמש
PUT Api/invitations/id/complate invitationId - עדכון סטטוס הזמנה ל-isFinish: true 400: חסר פרמטר invitationId, 404: הזמנה לא נמצאה, 500: שגיאה כללית
DELETE Api/invitations/id invitationId - מחיקת הזמנה 400: חסר פרמטר invitationId, 404: הזמנה לא נמצאה, 500: שגיאה כללית
POST Api/invitations userId, rooms - הוספת הזמנה 404: חסרים פרמטרים userId או rooms, 404: משתמש לא קיים, 404: חדר לא קיים, 400: חדר תפוס, 400: שגיאה כללית
